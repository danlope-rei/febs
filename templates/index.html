Quick Start

Default Entry Points

  • js: /src/entry.js
  • less: /src/entry.less>

Development bundles

Your development bundles are written to 

/src/dest/app.bundle.[js|css]

Production bundles

Your production bundles are written to 

/src/dest/app.bundle-<hash>.[js|css]

Commands 

$ febs dev
Builds development bundles
Bundles are unversioned and unminified
Sourcemaps are inline
$ febs prod
Builds production bundles
Bundles are versioned and minified
Sourcemaps are external.
$ febs dev-server
Run development server with live reloading of assets.
$ febs test
Run all of your unit tests (in /test)
